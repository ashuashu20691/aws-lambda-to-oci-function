Steps for Deployment Using Fn Project CLI

  1. Open a terminal and navigate to the directory containing the downloaded ZIP file.
  2. Extract the ZIP file contents.
  3. Run the following command to initialize the function: 
    fn init --runtime node oci-function
  4. Deploy the function to OCI Functions: 
    fn -v deploy --app myapp
  5. Validate and test the function deployment via the OCI Console or CLI.
